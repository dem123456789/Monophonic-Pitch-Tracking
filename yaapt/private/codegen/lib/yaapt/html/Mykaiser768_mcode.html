16% usage:  kaiser (n, beta)
17%
18% Returns the filter coefficients of the n-point Kaiser window with
19% parameter beta.
20%
21% For the definition of the Kaiser window, see A. V. Oppenheim &
22% R. W. Schafer, "Discrete-Time Signal Processing".
23%
24% The continuous version of width n centered about x=0 is:
25%
26%         besseli(0, beta * sqrt(1-(2*x/n).^2))
27% k(x) =  -------------------------------------,  n/2 <= x <= n/2
28%                besseli(0, beta)
29%
30% See also: kaiserord
31  
32% Author:  Kurt Hornik <Kurt.Hornik@ci.tuwien.ac.at>
33% Description:  Coefficients of the Kaiser window
34
35% 2000-02 Paul Kienzle (pkienzle@kienzle.powernet.co.uk)
36%    use besseli rather than jybess
37%    note, although Oppenheim & Schafer, 2nd edition has a formula
38%    which looks completely different than the one herein, it gives
39%    identical results
40  
41function w = Mykaiser (n)
42      beta = 0.5;
43  
44  if (n == 1)
45    w = 1;
46  else
47    m = n - 1;
48    k = (0 : m)';
49    k = 2 * beta / m * sqrt (k .* (m - k));
50    w = Mybesseli (0, k) / Mybesseli (0, beta);
51%     w_test = besseli (0, k) / besseli (0, beta);
52  end
53    
54end
